## 目录

| 路径 | 内容 |
|------|------|

| `product-src/` | 产品源码（HTML/CSS），改内容出新版用 |

## 快速开始（3步开卖）

1. 读 `从零到一销售手册.md`，按第1–2步搭号、开店
2. 把 `deliverable/` 里的 PDF 传网盘，商品设置自动发货
3. 用 `previews/` 的图 + `销售物料/` 的文案发笔记引流

## 出新版 / 改内容

```bash
# 编辑 product-src/planner.html（内容）或其中 :root CSS 变量（配色主题）

```
