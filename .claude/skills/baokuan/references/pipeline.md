# 爆款流水线（pipeline v1 — 2026-07-20）

> `/baokuan pipeline` 的作业书：从「选人 → 深度炼化 → 口播初稿 → 爆款化 → 豆包优化 → 事实终审」的全流程。
>
> **核心现实**：Claude 不能自动操作外部 AI 产品（Hermes、Nuwa/女娲、豆包）——它们由用户手动运行。
> 所以每个外部环节，**Claude 负责产出"投喂 prompt + 回收格式"，用户负责粘贴执行、把结果带回**。
> Claude 亲自做的是：**选人核实、口播爆款化、写豆包 brief、事实终审**。每次交接必须"无损"。

## 分工总览

| 阶段 | 谁做 | 输入 | 输出 |
|---|---|---|---|
| 0 选人 | **Claude**（radar+核实） | 100 Women 榜单 / radar / 用户给的名字 | 已核实「事实卡」 |
| 1 深度炼化 | **用户**（Hermes → Nuwa/女娲） | Claude 给的炼化 prompt | 该人物全部深度内容（逐字金句/方法/资产/反差） |
| 2 口播初稿 | **Claude**（baokuan v3.0） | 炼化产物 | 口播初稿 + 3 开头钩子 + 发布包 |
| 3 豆包优化 | **用户**（豆包专业版） | Claude 给的「豆包 brief」 | 抖音算法优化版口播 |
| 4 事实终审 | **Claude** | 豆包返回稿 | 核对后的最终发布包 + data-log 实验登记 |

## 阶段细则

### 0 选人（Claude）
- 来源：`/baokuan radar`（时效优先）/ 100 Women in AI 等榜单 / 用户直接给名字。
- 核实真人 + 事实，产出「事实卡」：姓名 / 身份反差 / 已核实数字（标出处）/ 可收藏资产 /
  代入锚点 / 是否有新闻引信 / 事实红线（哪些说法查无实据，禁用）。
- 过全部选题 gate（第1节保鲜期、第1节观众可用、v2.5 代入）。不过关不进下一步。

### 1 深度炼化（用户跑 Hermes + Nuwa/女娲）
- Claude 产出**炼化 prompt**，要求提取：逐字金句（原文+出处）、方法论步骤、免费可存资产、
  背景反差/逆袭线、争议点、可做续集的边角料。要求**保留每条的出处链接**。
- 用户把 Hermes/Nuwa 的结果带回。
- ⚠️ 炼化产物里的"事实"**仍是待核实**——AI 炼化会脑补，阶段 2 和 4 都要回原始出处核对。

### 2 口播初稿（Claude，套 baokuan v3.0 整台机器）
- 套 **Alisa 母版骨架**（具体数字连珠炮/反转/时效→细节→代入落中段→金句）。
- 过全部 gate：代入锚点(v2.5)、干货密度每15秒一条(v2.4)、六道工序(v2.7)、
  两级漏斗(v3.0：2秒钩子+中段留存)、事实红线、账号三承诺(最新/干货/真能用)。
- 输出：口播初稿 + 3 个开头 A/B + 中文标题/封面/标签/置顶/预埋回复 + 实验声明。

### 3 豆包优化（Claude 写 brief → 用户跑豆包专业版）
Claude 产出「豆包 brief」，硬性要求豆包在**不改事实**的前提下按抖音算法优化：
1. **2 秒跳出 <35%**：第一帧/第一句强钩子（具体数字 / 反转 / 冲突提问）。
2. **干货留扣子**：不全盘托出，边给边留"后面还有更狠的一条"，逼看完（对治中段漏水）。
3. **合时宜的幽默**：干、聪明、点到即止的操盘手式松弛——**不许滑稽/尬/损**，
   **不许冲淡代入情绪和金句**。幽默是调味，不是主菜。
4. **抖音算法友好**：完播、收藏钩、评论钩。
5. **守住账号内核**：英文口播 + 女性成长 + 操盘手调性 + 三承诺。
6. **红线告知豆包**：附上事实卡里"禁用的说法"，明令豆包不得添加任何新数字/头衔/事件。

### 4 事实终审（Claude）——⚠️ 整条流水线的保险丝，不能省
- 豆包**不知道哪些事实是核实过的**，最容易在这一步混入脑补/夸大（前科：Alisa被拒40次、120次拒绝）。
- 回收豆包稿后，**逐条比对阶段 0 的事实卡**：豆包新增/改动的任何数字、头衔、事件、去向，
  **没出处的一律删掉或改回**。软化未证实的精确数（如"211k"→"17.7万IG单平台"）。
- 定最终稿 + 在 data-log 登记本条实验变量（如"首条走完整pipeline+豆包优化"）。

## 铁律
1. 外部工具 Claude 不能代按，但每次 handoff 必须无损：给全 prompt、指定回收格式。
2. **事实红线贯穿全程；豆包那关之后必做终审**（最容易在这步混入脑补）。
3. 幽默是加分项不是主菜，绝不牺牲代入和干货。
4. 流水线只是"传送带"，判断标准仍是 playbook 全部 gate——传送带不降低质量门槛。
