# 质量门槛判断标准(入库前逐条对照)

四条同时满足才入库:
1. **反直觉或不为大众所知** — 拒收:"多喝水""要自律""建立自信"类常识;拒收任何没有具体数字、机制或专业圈内幕的泛泛之谈。
2. **有机制或数据支撑** — 必须能说清"为什么成立",来源等级从高到低:同行评审论文 > 权威机构(政府、学会、监管机构)> 行业报告(McKinsey/Pew 等)> 一线从业者实名公开分享。匿名帖子、自媒体转述不算硬来源。
3. **可操作** — 读者看完知道下一步做什么,必须有具体动作(话术、数字阈值、检查清单),不是"要注意""要重视"。
4. **与女性提升直接相关** — 通用建议必须有女性特殊角度或性别差异数据才收。

## 判例(Round 1 已裁决,后续保持一致)
- **门槛4(女性相关)是硬门槛**:来源再硬、机制再好,只要没有女性特殊角度或性别数据就丢弃。已丢弃判例:Lp(a) 检测(通用心血管知识)、SEC 费率侵蚀(通用理财知识)。"女性也适用"不等于"与女性直接相关"。
- **agent 自评"最弱"的条目默认丢弃**:子 agent 的来源可靠性说明里主动标弱的(如猎头佣金条,来源仅行业指南站),按宁缺毋滥处理。
- **宽区间数字要软化**:如"9%–60%"这类跨研究大区间,入库时改为定性表述("显著偏高"),避免内容创作时把数字讲死。

## 已知反面模式(直接丢弃)
- 鸡汤:成功学口号、无来源的励志断言
- 来源是内容农场/营销号的"研究表明"
- 复述常识但包装成秘密("早起的人更成功")
