# 出网限制与链接验证方法(本环境硬约束)

## 现象
本执行环境的出网代理策略屏蔽 WebFetch/curl 对多数外部域名的直接抓取(返回组织级 403),包括 nih.gov、pubmed/PMC、acog.org、oup.com、nature.com、sciencedirect 等学术/机构域名。这不是链接失效。

## 可行的验证方法(三个子 agent 均已验证可行)
- WebSearch 可用:搜索引擎返回的 URL + 页面正文摘要,能确认页面真实存在且内容与声明一致
- 要求 agent:每个链接必须出现在真实搜索结果中,且返回摘要与条目声明逐一比对;禁止凭记忆拼 URL
- 交叉印证:同一数字在 ≥2 个独立来源(官方新闻稿 + 期刊页/媒体)出现才算强验证

## 对二次审计的含义
- 二次审计应换一个 agent、用不同的搜索词重新检索同一主张,核对数字是否一致(而不是重复点链接)
- 引用了精确百分比/金额的条目是重点抽查对象
- 最终交付时在 INDEX 里注明:链接经 WebSearch 摘要核验,未逐页打开全文
