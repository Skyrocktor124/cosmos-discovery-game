# Chroma Cosmos · 双宇宙

启动后进入门户，可选择两个宇宙：

## 内在宇宙 · 上帝视角系统

一个帮你找到思想盲点、挖出限制性信念并清除它们的自我觉察工具：

1. **倾诉** — 写下困扰你的事、反复出现的模式、或纠结的决定（数据仅存在浏览器 localStorage，不上传服务器）。
2. **升维扫描** — Gemini 扮演「俯瞰者」，以第三人称俯瞰视角输出结构化报告：
   - **思维盲点**：认知偏误 + 你的原话作为证据 + 一个苏格拉底式反问
   - **限制性信念**：信念原文、可能来源、付出的代价、AI 建议的改写与 24 小时微行动
   - **视角切换**：十年后的你 / 完全的局外人 / 智慧导师
3. **信念星图** — 每条限制性信念是一颗持久化的「暗星」，跨多次倾诉累积成你的内在星图。
4. **清除仪式** — 四步流程（直视 → 审判 → 改写 → 释放）：必须亲手写下反驳证据和新信念，完成后暗星被点亮，觉察度上升。

> 它是自我觉察工具，不构成医疗或心理诊断，不能替代专业心理咨询。

## 外在宇宙 · 星际探索

原 Chroma Cosmos 玩法：程序生成的星区，由 AI 创造行星、恒星与星云。

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
